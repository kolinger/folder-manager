folder-manager
==============

Simple folder manager based on Nette Framework and Twitter Bootstrap

Usage
-----
1. download sources
2. make folders `/log` and `/temp` writtable
3. rename `/app/config/config.local.neon.example` to `/app/config/config.local.neon` and configure it
4. done :)
